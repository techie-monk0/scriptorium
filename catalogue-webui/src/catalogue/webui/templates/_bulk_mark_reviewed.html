{# Shared bulk "✓ Mark all reviewed" action for the review worklists (Books single + multi, Works-needing-review). The page turns on `bulk_select` (so the shell renders the select-all bar + per-row checkboxes) and defines `window.BB_REVIEW` with the per-id review endpoint; this script fills the shell's #bb-bulk-actions host and loops the ticked rows, retiring each through the SAME bbResolveRow the single-item button uses (so a bulk-reviewed row greys, badges ✓ and keeps its details exactly like a one-at-a-time review). Persons get their bulk "mark reviewed" through the picker's own server-side bulk bar (picker.html) instead, so they don't include this. #}
